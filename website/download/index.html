FAI - Fully Automatic Installation

Package download

The FAI software is divided in several packages:
  • fai-server: Package needed on the install server
  • fai-client: This package is needed on all hosts
  • fai-doc: The documentation
  • fai-nfsroot: Files only needed inside the nfsroot
  • fai-setup-storage: Contains the setup-storage tool
  • fai-quickstart: Meta package for easy setup of an install server
  • Entry for /etc/apt/sources.list and for /etc/fai/apt/sources.list: 

    deb http://fai-project.org/download squeeze koeln

  • The packages are signed with the gpg key of Thomas Lange. This is how you can get the gpg key in your apt configuration 
    gpg -a --recv-keys AB9B66FD; gpg -a --export AB9B66FD | apt-key add -

  • FAI packages from official project repository or from a Debian mirror.

Packages for Ubuntu

Have a look at PPA or the list of FAI packages in Ubuntu releases. 

deb http://ppa.launchpad.net/fai/ubuntu lucid main

FAI 3.4.x packages are included in Ubuntu maverick. 

deb http://archive.ubuntu.com/ubuntu maverick universe

Experimental packages

  • The experimental builds contain patches contributed by the developers that may or may not make it into the official version; the more feedback we get on this one, the likelier it is :-) (see also the announcement on linux-fai-devel) 

    deb http://fai-project.org/download experimental koeln
