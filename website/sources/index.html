FAI - Fully Automatic Installation

FAI Sources

The FAI software is divided into several packages:
  • fai-server: Package needed on the install server
  • fai-client: This package is needed on all hosts
  • fai-doc: Contains the documentation
  • fai-nfsroot: Package only needed inside the nfsroot
  • fai-quickstart: Metapackage for easy setup of an install server
  • Source Code via http
  • Checkout the sources: 

    svn co svn://svn.debian.org/svn/fai/trunk fai-sources

  • svn commit mails
