If you want to find all files (except hostnames) that are classes try:
   find /usr/local/share/fai -type f | egrep '/[[:upper:][:digit:]_]+$'

In cfengine scripts you can find class names using:
   egrep -r '[[:upper:][:digit:]_]+::' /usr/local/share/fai

Shell scripts are using the commad ifclass. So use:
   egrep -r 'ifclass[[:space:]]+[[:upper:][:digit:]_]+' /usr/local/share/fai


Class name	Description
------------------------------------------------------------

4GB		disk layout for disk up to 4.5 GB
ATA33		select UltraDMA mode2 on disk hda
BOOTP_SERVER	packages for a BOOTP server
COMPILE		packages for software development
DEBIAN_DEVEL	packages for a Debian developer
GERMAN		packages for the german language
NFS_SERVER	add user space nfs server to package list
NONIS		dont't use NIS
NIS		add NIS software, and configure NIS (nsswitch.conf,
		defaultdomain,host.conf.yp.conf )
NTP		add ntp software, create ntp.conf
SERVER		long list of packages, export /home, /usr to
		@faiclients, add tcsh to /etc/shells
KERNEL_SOFT	kernel sources and related packages

BASE		set UTC, longer interval for marks in syslogd, set
		passwd for root account, add account roott with same pw

HOME_CLIENT	add nfs mount $hserver:/home to fstab
DATALESS	add nfs mount $bserver:/usr to fstab, make /vol
		symlink, extract /usr/local/vol archive if available
USR_LOCAL_MOUNT add nfs mount $bserver:/usr/local to fstab
SCRATCH		export /scratch to netgroup @sundomain and @faiclients
FILES_SCRATCH	export /files/scratch to netgroup @sundomain and @faiclients
BOOT		create lilo.conf, call lilo
FAI_BOOTPART	create FAI boot partition, add it to lilo.coonf
NET		create eth0 configuration, resolv.conf, copy ssh_known_hosts
NETWORK		disable fingerd, tald, ntalkd, rexecd, discard,
		daytime, identd in inetd.conf
NOTEBOOK	increase max count between two fsck for all partitions
NET_9		don't use tcp wrapper for rsh, rlogind, rstatd, rusersd

LAST		remove backup files of cfengine, link /tmp partition
		if necessary, add nfs location of debian mirror to fstab
DEFAULT		minimal software list, set timezone, hostname, create
		lo device configuration, keymap copy dot files for
		root account, make /dev/fd* writeable for all, install
		kernel package

CS_KOELN	template of dot files for our institute
S3VIRGE		XF86 configuration for our S3 virge graphic cards



Following classes should always be define:

LAST, DEFAULT (are defined by default)
BASE
BOOT
NET

Some classes depends on another class. For eg. the class NTP is used
in script NET. So both classes must be defined, if the ntp
configuration should be created.
